- CSS 선택자 (selector) 
  html에 스타일을 적용할때 
  html 요소를 지정하는 역할을 한다. 

  내가 작성한 모든 태그에 스타일을 적용하려면
  일일이 태그를 적어야 하는가? 

  - 전체 선택자  * 
  - 유형 선택자  h1 p span 
  - 아이디 선택자  : 해당 식별된 태그에 대해서만 골라서 적용

    클래스명과 일치하는 요소를 선택하고, 하나 이상 스타일을 지정할 수 있다.

2. CSS 배치 속성 : 원하는 위치에 요소를 배치할 수 있도록 레이아웃 속성, 마진, 테두리, 패딩 .... 박스모델 (BOX MODEL)
   반응형 디자인이 무엇인지?
   많은 사이트가 다양한 화면에 맞춰 자동으로 요소를 배치하는 웹페이지를 반응형 웹이라고 합니다.
   - 기기 화면에 맞춰 요소가 자동으로 배치된 웹 페이지를 제공하므로, 기기화면마다 따로 코드를 개발 관리하지 않아도 되며 유지관리도 용이함.
   - 개발 비용을 절약할 수 있으며 사용자 편의성을 높일 수 있다.

배치 속성이란?
CSS 레이아웃

모든 HTML요소에는 display 속성이 있다.
display 속성의 기본값은 블럭 또는 인라인 설정되어 있다.

블록레벨 요소는 요소(태그)가 가로 줄 전체를 차지한다. <h1>
인라인 레벨 요소는 요소가 있는 공간만을 차지한다. <span>

태그마다 이 두 특성은 고정일까?
display라는 CSS 속성으로 바꿔줄 수 있다.

블록과 인라인이라는 개념은 왜 필요할까?
보기 좋게 배치하기 위해서 필요하다.
원하는 위치에 요소를 배치하려면, 태그마다 가지고 있는 특성을 알아야한다.

블럭과 인라인 요소 그리고 레이아웃을 정하는 HTML 요소가 바로 박스모델이다.

CSS 박스 모델은 기본적으로 모든 HTML을 감싸는 상자이다.
- 요소(content) : 텍스트, 사진 등 보여줄 대상
- 패딩(padding) : 요소 주변 영역을 감싼다. 해당 부분은 투명색이다.
- 테두리(border) : 요소와 패딩을 감싸는 테두리이다.
- 마진(margin) :  테두리 밖의 영역을 감싼다. 투명색이다.
